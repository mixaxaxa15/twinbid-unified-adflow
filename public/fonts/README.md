# Шрифты прелендов

Inter, Roboto, Montserrat, Open Sans, Oswald, Playfair Display, Bebas Neue,
Russo One, Ubuntu и PT Sans поставляются локально через пакеты Fontsource.
Оригинальные лицензии каждого семейства находятся в соответствующем пакете
`node_modules/@fontsource/<family>/LICENSE`.

Загруженные пользователем файлы сохраняются в каталоге `custom` и попадают в
экспорт конкретного преленда.
